## Theobot

[![GitHub Actions status](https://github.com/lmichelin/theobot/workflows/CI/badge.svg)](https://github.com/lmichelin/theobot/actions)

A Slack bot for Theodo to help new theodoers breaking the ice.

## Installation

```bash
$ yarn
```

## Running the app

```bash
# development
$ yarn start

# debug mode
$ yarn start:debug

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
